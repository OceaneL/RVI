TP6

Partie 1:

Voir la scene tp6-camera
script CameraDualPoint (attaché à la camera)

Partie 2:

Voir la scene tp6-manip
script interaction souris (attaché au game manager)
rester appuyer sur le bouton gauche de la souris pour se déplacer sur le plan x,z.
rester appuyer sur le bouton droit de la souris pour se déplacer sur le plan parallèle à l'écran.

Partie 3.1

Voir la scene tp6-navig-vitesse 
script navigation fps (attaché à la camera)
Navigation FPS: clavier haut/bas : avancer reculer dans le plan X-Z
		clavier gauche/droite : se déplacer à gauche ou à droite
		souris axe X + click bouton gauche : tourner le torse gauche/droite
		souris axe Y + click bouton gauche : tourner la tête haut/bas
Navigation contrôle en vitesse (plus la souris pressé s'éloigne du point initial du click, plus le déplacement d'effectue rapidement):
		souris axe Y + click bouton droit : avancer reculer dans le plan X-Z
		souris axe X + click bouton droit : à gauche ou à droite

Partie 3.2

Voir la scene tp6-navig
script navigation cible (attaché à la camera)
navigation et rotation avec le click gauche de la souris.
